## 오름차순
```Java
public class SortArrayList {
    public static void main(String[] args) {
 
        // 오름차순으로 정렬
        Collections.sort(list);
 
    }
}
```

## 내림차순
```Java
public class SortArrayList {
    public static void main(String[] args) {
 
        // 내림차순으로 정렬
        Collections.sort(list, Collections.reverseOrder());
        
    }
}
```
